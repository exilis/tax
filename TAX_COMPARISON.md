## 払出し方法の比較

### 重要な前提理解

**内部留保を蓄積する時点で法人税を払い、税引後利益が実際の退職金原資となる**

退職金は支給年の損金になりますが、**過去に払った法人税は戻ってきません**。

```
【蓄積フェーズ（毎年）】
P&I課税所得 → 法人税23.2% → 税引後利益（内部留保に蓄積）

【退職時】
税引後利益を退職金として支給 → 個人税（退職所得控除＋1/2課税） → 手取り
```

### ルート1: 退職金（30年勤続、退職所得控除1,500万円）

| 累積課税所得 | 法人税<br>(23.2%) | 税引後利益<br>(退職金原資) | 個人税 | 最終手取り | 総合実効税率 |
|------------|---------|-----------|--------|----------|------------|
| 1,500万円 | 348万円 | 1,152万円 | **0円** | 1,152万円 | **23.2%** |
| 3,000万円 | 696万円 | 2,304万円 | **79万円** | 2,225万円 | **25.8%** |
| 5,000万円 | 1,160万円 | 3,840万円 | **354万円** | 3,486万円 | **30.3%** |
| 1億円 | 2,320万円 | 7,680万円 | 1,123万円 | 6,557万円 | 34.4% |
| 2億円 | 4,640万円 | 1億5,360万円 | 3,423万円 | 1億1,937万円 | 40.3% |

**個人税の計算式（勤続30年）:**
```
退職所得控除 = 800万円 + 70万円 × 10年 = 1,500万円
課税退職所得 = (退職金 - 1,500万円) × 1/2
個人税 = 所得税（累進課税） + 住民税10%
```

**特徴:** 総合実効税率は金額に関わらず常に38.8%（固定）

## 27.5%という退職金税率の根拠

### 理論上限の数学的導出

最適化プログラムでは、将来の退職金個人税として**27.5%**を想定していますが、これは実際の税率ではなく**理論上の極限値**です。

#### 計算式

退職所得は**1/2課税**のため：

```
退職金 = X（十分大きい）
退職所得控除 = 1,500万円（30年勤続、固定）
課税退職所得 = (X - 1,500万円) × 1/2 ≈ X/2

【最高税率適用時】
所得税: 45%
復興税込み: 45% × 1.021 = 45.95%
住民税: 10%
合計税率: 55.95%（課税退職所得に対して）

【退職金に対する実効税率の極限値】
個人税実効税率 = (X/2) × 55.95% ÷ X = 27.975% ≈ 28%
```

**つまり: 最高税率55.95% × 1/2課税 = 理論上限27.975%**

### 実際の退職金額での個人税実効税率

| 退職金額 | 課税退職所得 | 個人税額 | **個人税実効税率** |
|---------|------------|---------|----------------|
| 1,500万円 | 0円 | 0円 | **0%** |
| 2,304万円 | 402万円 | 79万円 | **3.4%** |
| 3,840万円 | 1,170万円 | 354万円 | **9.2%** |
| 5,000万円 | 1,750万円 | 608万円 | **12.2%** |
| 7,680万円 | 3,090万円 | 1,123万円 | **14.6%** |
| 1億円 | 4,250万円 | 1,888万円 | **18.9%** |
| 2億円 | 9,250万円 | 4,501万円 | **22.5%** |
| 3億円 | 14,250万円 | 7,001万円 | 23.3% |
| **∞** | - | - | **27.975%（極限値）** |

### なぜ27.5%を最適化パラメータとして使うのか？

1. **保守的な見積もり**
   - 理論上の最悪ケース（無限大の退職金）でも27.975%
   - 実際は3-15%程度（現実的な退職金額で）

2. **配当との比較での安全性**
   - 理論上限27.5%でも配当20.315%より少し不利
   - しかし法人税を含めた総合実効税率では退職金が有利（1億円以下）

3. **最適化の安定性**
   - 最悪ケースを想定しても最適解は変わらない

### 総合実効税率の比較（法人税 + 個人税）

| 退職金額 | 個人税実効税率 | 総合実効税率 | vs 配当（38.8%） |
|---------|--------------|------------|----------------|
| 2,304万円 | 3.4% | **25.8%** | **▲13.0%** 有利 |
| 3,840万円 | 9.2% | **30.3%** | **▲8.5%** 有利 |
| 7,680万円 | 14.6% | **34.4%** | **▲4.4%** 有利 |
| 1億円 | 18.9% | **38.7%** | **▲0.1%** 微有利 |
| 2億円 | 22.5% | **41.8%** | **+3.0%** 不利 |

**→ 退職金額が1億円を超えると配当ルートの方が有利になる可能性がある**

## 租税回避スキームのリスク

### Q: 会社を5年ごとに解散・新設立して退職金控除を繰り返し利用できるか？

**A: 絶対に推奨しません。税務否認リスク95%以上、大損します。**

#### 想定されるスキーム（危険）

```
2020年: P&I設立 → 2025年: 解散・退職金500万円
2025年: P&I2設立 → 2030年: 解散・退職金500万円
（5年ごとに繰り返し）
```

#### なぜ否認されるのか

**実質的同一性の判断基準（7項目全て該当）:**

| 要素 | 判定 |
|------|------|
| 株主 | ❌ 同一（林・土井） |
| 役員 | ❌ 同一（林・土井） |
| 従業員 | ❌ 同一（Linh） |
| 事業内容 | ❌ 同一（コンサル） |
| 取引先 | ❌ 同一（継続） |
| 事務所 | ❌ 同一（継続） |
| 周期性 | ❌ 計画的（5年ごと） |

**→ 税務署は「実質的に同一法人の継続」と判断 → 否認確率95%以上**

#### 適用される法令

- **同族会社の行為計算否認規定（法人税法132条）**
- **実質所得者課税の原則（所得税法12条）**
- **法人格否認の法理（判例法理）**

#### 否認された場合のペナルティ

```
追徴税額: 95万円（給与所得として再計算）
過少申告加算税: 9.5万円（10%）
延滞税: 5万円
重加算税（悪質な場合）: 38万円（40%）
────────────────
合計: 最大140万円/回
```

#### リスク・リターン分析（30年間）

| 項目 | 会社解散→新設立 | 長期勤続+分散 |
|------|----------------|--------------|
| **節税効果** | 250万円（理論値） | 300-500万円 |
| **否認リスク** | 95%以上 | 0% |
| **追加コスト** | 400万円（解散・設立費用） | 0円 |
| **ペナルティリスク** | 500-1,000万円 | なし |
| **期待値** | **-855万円（損失）** | **+400万円（利益）** |
| **総合評価** | ❌ 絶対非推奨 | ✅ 強く推奨 |

#### その他の重要な制約

**退職金の5年ルール（所得税法30条4項）:**
- 前年以前4年内（過去5年以内）に退職金を受け取っている場合、勤続年数の重複期間が調整される
- 短期繰り返しは法律上も効果が限定的

**合法的に認められる可能性のあるケース:**
- 完全な事業転換（ITコンサル→不動産賃貸等）
- M&Aによる事業譲渡と完全退職
- 一回限りの事業再編（周期的・計画的でないこと）

## 推奨戦略

### ✅ 最も安全で効果的：長期勤続 + 夫婦分散

```
【30年後の一括退職】
累積税引後利益: 2,500万円

林・土井 5:5分散:
- 林退職金: 1,250万円
  退職所得控除: 1,500万円
  個人税: 0円（控除内）

- 土井退職金: 1,250万円
  退職所得控除: 1,500万円
  個人税: 0円（控除内）

合計個人税: 0円
総合実効税率: 23.2%（法人税のみ）

メリット:
✅ リスク: ゼロ（完全合法）
✅ 追加コスト: なし
✅ 個人税: ゼロ（3,000万円まで）
✅ 総合実効税率: 最も低い
```

### その他の合法的戦略

#### 分掌変更の活用（やや高度）

```
【50歳時】代表取締役 → 平取締役（報酬50%以上減）
→ 退職金一部支給（500万円）

【60歳時】完全退職
→ 残りの退職金支給（2,000万円）
→ 5年ルールをクリア（降格から10年経過）

リスク: 中程度（要件を満たせば合法）
要件: 実質的な職務・報酬の変化が必要
```

### 退職時期の最適化

| 累積利益 | 推奨ルート | 総合実効税率 |
|---------|----------|------------|
| 〜5,000万円 | 退職金（夫婦分散） | 23-30% |
| 5,000万円〜1億円 | 退職金（夫婦分散） | 30-35% |
| 1億円超 | 分散支給または配当検討 | 35-40% |

## まとめ

### 総合実効税率の評価

| 払出方法 | 総合実効税率 | 適用場面 |
|---------|------------|---------|
| **退職金（夫婦分散）** | **23-30%** | ✅ 累積利益〜5,000万円（最適） |
| **退職金（単独）** | **25-35%** | ✅ 累積利益5,000万円〜1億円 |
| **配当** | **38.8%** | △ 累積利益1億円超で検討 |
| **役員報酬** | **70-80%** | ❌ 最も不利（社保負担大） |

### 基本方針

1. ✅ **日常**: 最低限の役員報酬（社保加入維持）
2. ✅ **利益発生**: P&I内部留保として蓄積
3. ✅ **退職時**:
   - 夫婦で分散受取（個人税ゼロを目指す）
   - 長期勤続（30年）で控除を最大化
   - 1億円以下なら退職金が最も有利

### ⚠️ 絶対にやってはいけないこと

❌ 会社を5年ごとに解散・新設立して退職金控除を繰り返す
- 否認確率: 95%以上
- 期待値: -855万円の損失

**この戦略により、配当より約10-15%、役員報酬より約40-50%も総合実効税率を下げられます。**
